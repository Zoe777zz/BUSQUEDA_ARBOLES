# BUSQUEDA_ARBOLES
# Taller 9: Implementación de Algoritmos de Recorrido de Árboles
Este proyecto contiene la implementación en Java de los algoritmos fundamentales para el recorrido de estructuras de datos tipo árbol, desarrollado como parte de la carrera de Computación.

# Objetivos

•	Implementar los recorridos Preorden, Inorden y Postorden mediante recursividad.
•	Implementar el recorrido Por Niveles utilizando una estructura de datos tipo cola.
•	Verificar los resultados mediante ejecución en consola y comparación manual.

# Guía de Instalación y Uso
# Requisitos Previos

Antes de comenzar, asegúrate de tener instalado lo siguiente:
•	JDK: OpenJDK (obligatorio para este taller).
•	IDE: Visual Studio Code (con Extension Pack for Java) o IntelliJ IDEA.
•	Git: Para la gestión del repositorio.
# 1. Clonar el repositorio

Abre una terminal en tu computadora y ejecuta el siguiente comando para obtener una copia local del proyecto:

https://github.com/Zoe777zz/BUSQUEDA_ARBOLES.git

# 2. Abrir el proyecto

•	Si usas VS Code: Abre la carpeta del proyecto y asegúrate de que el "Language Support for Java" esté activo.
•	Si usas IntelliJ: Selecciona Open y busca la carpeta donde clonaste el repositorio.

# 3. Ejecución

Localiza la clase principal (donde se encuentra el método main) y ejecútala. El programa mostrará en consola:
1.	El árbol de prueba construido.
2.	El orden de visita de los nodos según cada algoritmo.
